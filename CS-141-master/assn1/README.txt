File name with source code is nearest_neighbor.py. The program takes in a text file from the command line. 
Run the program by typing in: "python nearest_neighbor.py example.txt" where example is the name of the
txt file that you want to use. The output file is named "exampledistance.txt" where example is the name of 
the input file you passed in via the command line. The output file contains the distance of the closest pair
output on one line.
